# Fox Valley Music Foundation

This is the Fox Valley Music Foundation website built with:

- **Next.js 15** - React framework
- **React 19** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Sanity CMS** - Content management
- **Vinyl Revival Aesthetic** - Bold, distinctive design

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3001
```

## Vinyl Aesthetic

The site features a unique "Vinyl Revival Meets Modern Activism" design:

- **Typography**: Bebas Neue (display), IBM Plex Sans (body), IBM Plex Mono (metrics)
- **Colors**: Blue Note Blue, Motown Gold, Capitol Orange
- **Unforgettable Element**: Spinning vinyl records that reveal student photos on hover
